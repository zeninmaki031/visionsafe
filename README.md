 Color Accessibility Tool

A lightweight, single-file web tool designed to help designers create color combinations that are accessible to people with color vision deficiencies.

About the Project

This project focuses on promoting inclusive design by making it easier to choose colors that work for everyone, including users with color blindness. Built with simplicity in mind, it demonstrates how even small tools can improve accessibility.

Features
Simple and easy-to-use interface
Lightweight (built with a single HTML file)
Helps create accessible color combinations
Focused on inclusive and user-friendly design
How to Use
Download or clone this repository
Open the HTML file in your browser
Start experimenting with colors
 Why This Project?
Design should be inclusive. This tool aims to raise awareness and provide a simple solution for designers to make better, more accessible color choices.
